# Club Simulation

This project simulates the behavior of patrons in a virtual club environment.

## Features

- Simulates clubgoers' movement, interactions, and activities.
- Includes a bartender, and clubgoers.
- Supports pause and resume functionality.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher.

### Installation

1. Compile the code: make
2. Run the simulation: make run
3. To specify parameters: make run ARGS="arg1 arg2 arg3 arg4"

Alternatively:

2. Compile the code: `javac clubSimulation/*.java`
3. Run the simulation: `java clubSimulation/ClubSimulation`

where arg1 : gridX
      arg2 : gridY
      arg3 : number of Clubgoers
      arg4 : max people in club	

## Usage

1. Run the simulation.
2. Observe the interactions and movements of clubgoers and staff.
3. Use the pause button to pause the simulation and resume button to resume.
